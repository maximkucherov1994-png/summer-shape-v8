# Summer Shape Maxim V10

## Wichtig
Für Vercel darf nicht nur eine ZIP-Datei hochgeladen werden. Im GitHub-Repository müssen die Projektdateien sichtbar sein:
app / lib / public / package.json / next.config.js usw.

## Upload zu GitHub auf Windows
1. ZIP entpacken.
2. Den entpackten Ordner öffnen.
3. Den Inhalt auswählen: app, lib, public, package.json, next.config.js usw.
4. In GitHub über Upload files hochladen.
5. Commit changes.

## Vercel
1. Add New → Project.
2. Repository auswählen.
3. Framework: Next.js.
4. Deploy.
