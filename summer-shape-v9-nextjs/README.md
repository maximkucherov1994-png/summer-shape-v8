
# Summer Shape V9

## Upload zu GitHub
1. Öffne dein Repository `summer-shape-v8`.
2. Klicke auf `uploading an existing file`.
3. Lade alle Dateien/Ordner aus diesem Projekt hoch.
4. Commit changes.

## Vercel
1. Öffne Vercel.
2. Add New → Project.
3. GitHub Repository `summer-shape-v8` auswählen.
4. Framework Preset: Next.js.
5. Deploy klicken.

## iPhone
1. Vercel-Link in Safari öffnen.
2. Teilen-Button.
3. Zum Home-Bildschirm.
4. Hinzufügen.
